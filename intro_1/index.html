what is MySQL? - database system ระบบฐานข้อมูลที่จะใช้บนเว็บ ทำงานบนserver - ฐานข้อมูลเอาไว้เก็บข้อมูล เช่น เว็บบริษัท - Employees พนักงาน - Products สินค้า - Customers ลูกค้า - Order การสั่งซื้อสินค้า - ภาษา MySQL จะใช้ร่วมกับภาษา PHP - PHP เป็นภาษาที่ทำงานบน Server เหมือนกับ MySQL - โปรแกรมที่จำเป็นต้องใช้คือโปรแกรม Xampp เปิด start Apache กับ MySQL
